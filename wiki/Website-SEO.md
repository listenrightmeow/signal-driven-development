# Website SEO

The SDD website targets being the definitive search result for SDD-related queries. SEO is built into the architecture, not bolted on.

## Technical SEO

### Semantic HTML

- Proper heading hierarchy (single `<h1>` per page)
- Semantic elements: `<article>`, `<section>`, `<nav>`, `<main>`
- Clean URL structure: `/walkthrough`, `/gap-categories`, `/get-started`
- No hash fragments for primary content

### Structured Data (JSON-LD)

| Schema | Page | Purpose |
|--------|------|---------|
| `Article` | Homepage | Methodology overview |
| `HowTo` | Get Started | Quick-start guide |
| `BreadcrumbList` | All pages | Navigation |
| `Organization` | All pages | Authorship |

### Meta Tags

- Unique `<title>` and `<meta name="description">` per page
- Open Graph tags for social sharing
- Twitter Card tags
- Canonical URLs on every page

### Technical

- `sitemap.xml` auto-generated by Astro sitemap integration
- `robots.txt` permissive
- Static HTML -- fastest possible crawl and index
- Sub-second FCP directly supports page speed ranking signal
