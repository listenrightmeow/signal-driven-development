# Changelog

All notable changes to the Signal-Driven Development methodology will be documented in this file.

This project uses methodology versioning:

- **MAJOR**: Breaking changes to templates, gap categories, or the convergence model
- **MINOR**: New templates, examples, fields, or documentation sections
- **PATCH**: Typo fixes, clarifications, editorial improvements

## [1.0.0] - 2026-03-20

### Added

- Four core templates: domain specification, gap report, gap resolution log, architecture palette
- Four gap categories: Structural (SG), Heuristic (HG), Language (LG), Decision (DG)
- Three-pass convergence model with measurable definition of done
- Worked example: Greenfield Veterinary Clinic (3 passes, 18 to 0 gaps)
- Severity model: ERROR and WARNING classifications
- Heuristic thresholds sourced from Evans, Vernon, Brandolini, and Khononov
